# Overview

AIMES Ecosystem API allows you to programmatically retrieve data and predictions that are made available in the AIMES Ecosystem Platform.
This is useful when you want to integrate our dataset into your system or workflow.

### About the APIs

You can get programmatic access to the following capabilities:
1. Retrieve road network layout (Links, Intersections, and the connections).
2. Retrieve actual and predicted 15-minute speed for all Links.
3. Retrieve actual and predicted 15-minute volume for all Links.
4. Retrieve estimated and predicted congestion for all Links.
5. Retrieve estimated and predicted hazard for all Links.
6. Retrieve predicted Link emission for all Links.
7. Retrieve estimated and predicted queue length for all Intersections.
8. Retrieve estimated signal cycle time for all Intersections.
