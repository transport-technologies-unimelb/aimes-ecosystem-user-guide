# Limits

API usage is limited to the following usage limits:

<table>
  <tr>
    <th align="left">Total number of invocations</th>
    <td>There is no limit on the total number of invocations.</td>
  </tr>
  <tr>
    <th align="left">Rate limit</th>
    <td>15,000 requests per 5 minute for each unique IP address.</td>
  </tr>
  <tr>
    <th align="left">Regions</th>
    <td>Follows the region limitation imposed to your user account.</td>
  </tr>
  <tr>
    <th align="left">Features</th>
    <td>Follows the feature limitation imposed to your user account.</td>
  </tr>
</table>
